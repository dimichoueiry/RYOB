# Validator - Startup Idea Validation Tool

A high-impact landing page for a startup idea validation tool.

## Tech Stack
- **Vite**: Fast development server and bundler.
- **Vanilla JS**: Lightweight interactions and animations.
- **CSS Variables**: Easy theming (Dark Mode).

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open your browser at `http://localhost:5173`.

## Project Structure
- `index.html`: Main landing page structure.
- `src/style.css`: All styles, including animations and responsive design.
- `src/main.js`: Logic for scroll animations and dynamic elements.
